Detailed Description

template<>
class hive::misc::EndianT< 1234 >

The little-endian converter.

This class is used to convert little-endian integers to the host and visa versa.

Please use the h2e() static methods to convert integers from host to little-endian and e2h() static methods to convert integers from little-endian to host.
