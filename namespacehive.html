typedef boost::noncopyable NonCopyable

The non-copyable interface.

This class should be used as private base class to emphasise that object cannot be copied. It is especially useful for classes which holds some h/w resources and therefore cannot be copied.

class MyDevice:
private hive::NonCopyable
{
public:
int foo;
};
void f()
{
MyDevice a;
MyDevice b(a); // error!
b = a; // error!
}
Note
Anyway it is possible to explicit provide copy-constructor and/or assignment operator for derived classes.
