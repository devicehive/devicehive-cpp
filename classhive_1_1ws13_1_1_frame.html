Detailed Description

The WebSocket frame.

This class contains full formated frame:

field size in bytes
control 1
length 1 or 3 or 9
mask 0 or 4
payload N

The empty frame doesn't contain any data.

The following non-control payloads may be used:

  • Continue
  • Text
  • Binary as well as control payloads:
  • Close
  • Ping
  • Pong
