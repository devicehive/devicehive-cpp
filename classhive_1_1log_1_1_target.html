Detailed Description

The log target.

This is the base class for all log targets - endpoint for all log messages. Base class does nothing and works like a NULL target.

See the inner classes for standard targets: Stream, File and Tie.

The target uses specified log message format (see setFormat() method) or hive::log::Format::defaultFormat() if no any format provided.

There is a simple filter feature - you can provide the minimum logging level. All log messages with the logging level below this limit will be ignored. This simple message filter is disabled by default, i.e. the minimum logging level is hive::log::LEVEL_ALL.
