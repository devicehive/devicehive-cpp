Detailed Description

The code block logging.

Is used to log method calls or other code blocks. At construction sends ENTER log message. At destruction sends LEAVE log message.

For example:

void f()
{
HIVELOG_TRACE_BLOCK(logger, "f()");
// function body
}

Cannot be used with temporary strings.
