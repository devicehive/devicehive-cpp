enum StatusCode

The status codes.

Enumerator
STATUS_NORMAL 

Indicates a normal closure.

STATUS_GOING_AWAY 

Indicates that an endpoint is "going away".

STATUS_PROTOCOL_ERROR 

Indicates that an endpoint is terminating the connection due to a protocol error.

STATUS_UNEXPECTED_DATA 

Indicates that an endpoint is terminating the connection because it has received a type of data it cannot accept.

STATUS_INVALID_DATA 

Indicates that an endpoint is terminating the connection because it has received data within a message that was not consistent with the type of the message.

STATUS_MESSAGE_TOO_BIG 

Indicates that an endpoint is terminating the connection because it has received a message that is too big for it to process.

STATUS_NO_EXTENSION 

Indicates that an endpoint (client) is terminating the connection because it has expected the server to negotiate one or more extension.

STATUS_UNEXPECTED_COND 

Indicates that a server is terminating the connection because it encountered an unexpected condition that prevented it from fulfilling the request.
