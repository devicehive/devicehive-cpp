Detailed Description

The XBee frame.

This class contains full formated frame:

field size in bytes
signature 1
length 2
payload N
checksum 1

The empty frame doesn't contain any data.

There are a few payloads:

  • ATCommandRequest
  • ATCommandResponse
  • ZBTransmitRequest
  • ZBTransmitStatus
  • ZBReceivePacket
